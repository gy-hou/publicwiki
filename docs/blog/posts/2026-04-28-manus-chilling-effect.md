# Manus 带来的寒蝉效应

> 即使变成新jiapo主体，也不等于安稳下车。只要还有一点和 --内 有关，就可能被卡。

<!-- more -->

## 启示：AI 创业要在 Day 0 做身份二选一

以后 AI 创业从 0 开始时，就得尽早二选一。这里说的 0 不是产品，不是技术，而是**身份**。

Manus 倒霉的地方，在于它的 0 从一开始就是伪命题：想切割，但切不干净。

## 关于数据：A 是 A、C 是 C 的二分法并不成立

Manus 肯定用到过 --内 数据，但我们蒸馏别人时，也用了别人的数据。整个 AI 技术栈本来就是全球知识堆叠出来的：Transformer 来自 Google DeepMind，后续能力来自全球开源与闭源的共同迭代。

在这种现实里，简单按 “A 是 A，C 是 C” 去硬切，实际很难落地。

## 关于 shenhe：该审就审，但别情绪化扣帽子

外资并购本来就有成熟流程：可以审查、可以否决、可以撤销。

但把创业公司出海融资、出售股权直接扣成大帽子，太情绪化。非要低价卖给企鹅和字jie，创业者以后怎么活？

## 我最替 Manus 不值的点

我最不值的是：从一开始，边界就不清楚，这个锅不能全让它背。

某宝、pdd 都有外资，我们加入 WTO 本来就是要引进外资。  
如果后续认定它敏感、属于 zhanlue 资产，那就该给出更清晰、可执行的路径。

## 更合理的方案：按战略资产处理，就给合理价格

如果认为它是 zhanlue 资产，就让 guojia 队出手收购，给一个合理价位。

Meta 几十亿美元，字节给的最高几千万美元，大不了给个折中价格收购。  
重拳出击四个月，小厂以后还怎么敢创业？

## 我的立场

我坚决支持、肯定并认可官方操作，所以这里只做理性探讨，不输出情绪。

我担心的不是 Manus 本身，而是：小厂以后到底怎么创业。

## 现在的问题：边界不清，创业决策就会失真

现在大量小厂都在用 Claude、GPT、Gemini 和海外开源框架。AI 本来就是全球技术栈。做视频的、做内容创作的、做科研的，前段时间新加坡 claude 全球使用率第一，背后是什么大家都知道。

如果用了国外 AI 就不算自主，用了国内数据就不能出海，那边界到底在哪里？

现在这种不确定性，会把小厂逼到只剩两个选择：  
从 Day 0 彻底离岸，或者干脆别干。
