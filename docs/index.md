>
> 把小红书装不下的干货，都放在这里。

## 关于我

我是一个长期深度使用 AI 的实践者，也在小红书持续分享能直接复用的工作方法。  
我希望把分散在图片、评论区和对话里的经验整理成结构化知识，减少重复踩坑。  
作为开源爱好者，我会把能公开的模板、流程和思路持续沉淀在这里。  
这个站点会优先更新“可复制、可落地”的内容，而不是只讲概念。

!!! info "为什么做这个 Wiki"
    - 小红书有字数限制，很多干货只能做成图片。
    - 图片里的代码和 Prompt 没法复制，读者体验会被打断。
    - 我想把优质内容结构化、开源出来，方便持续迭代。
    - 这里的内容都尽量做到“看完就能复制使用”。
